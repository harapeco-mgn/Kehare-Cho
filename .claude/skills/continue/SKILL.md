# Continue Previous Work

前回の作業を継続します。

## Instructions

1. **最新のプランファイルを確認する**
   - `~/.claude/plans/` 配下の最新プランファイルを読む
   - Git履歴を確認して前回のコミットから作業内容を把握する

2. **ファイルの再読み込みを最小限にする**
   - 既に議論済みのファイルは再読み込みしない
   - 必要な場合のみ、変更が必要なファイルだけを読む

3. **即座に実装を開始する**
   - 調査フェーズや新しい計画ファイル作成をスキップする
   - プランの次のステップから実装を始める
   - 質問ではなく、コードを書くことに集中する

4. **実装完了まで進める**
   - テストを実行して結果を確認する
   - 必要ならコミットまで完了させる
   - 途中で止まらず、タスクを完遂する

## Rules

- ❌ 同じファイルを複数回読まない
- ❌ 計画だけ作って実装しないで終わらない
- ❌ ユーザーに細かい確認を何度も求めない
- ✅ 合理的な判断で自律的に進める
- ✅ 判断理由はコメントやコミットメッセージに残す
